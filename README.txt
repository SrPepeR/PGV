En el java MAIN están todas las pruebas, los datos iniciales van el el archivo datos.txt.
El programa genera el archivo ordenado.txt con todos los datos valido ordenados y separados por ;.
Añado las librerias por si acaso.
Son 6 clases:
	MAIN:			Con las pruebas.
	CualEsMayor:		Imprime en pantalla el número más grande de dos introducidos.
	PGVArrays:		Saca el mayor, menor, la suma y la media además de ordenarlo mediante burbuja.
	LecturaFichero:		Lee los números del fichero y devuelve un array con los números ordenados con PGVArrays.ordenaBurbuja(ArrayList<Integer>);
	EscribirEnFichero:	Escribe los número en el fichero "ordenado.txt".
	Grafico:		Crea el grafico utilizando JFrame y JFreeChart.
